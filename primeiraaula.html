Adobe DreamWeaver

Explique o que é e o que podemos fazer com o DreamWeaver

É um software de desenvolvimento web que permite criar, editar e gerenciar sites e aplicações web. Ele combina com uma interface visual com editor de códigos

Podemos criar sites, páginas da Internet, montar lojas virtuais, plataformas de notícias e outros diversos tipos de conteúdos digitais para colocar na Internet.

Como trabalhamos ou fazemos.

Temos duas opções de usarmos o DreamWeaver, podemos criar os conteúdos dos sites incorporando os elementos por itens nos Menus (Visual), ou criarmos o site programando (TAGs) com códigos em HTML, CSS e outras linguagens de programação Web.

Explique o HTML.

É uma abreviação de Hypertext Markup Language - Linguagem de Marcação de Hypertexto. Resumindo em uma frase: o HTML é uma linguagem para publicação de conteúdo digital pra Web (texto, imagem, vídeo, áudio, tabela, formulário e etc). Uma linguagem padronizada para a fácil navegação como um documento digital acessado pela Internet ou offline, definido por TAGs

O que são as TAGs, descreva exemplos.

Composto por Comandos ou Rótulos (Etiquetas) que serão interpretadas pelos Navegadores, apresentando o conteúdo de forma gráfica.

Explique o CSS.

Cascading Style Sheets, significa Folha de Estilo em Cascata. Ele é parte integrante do HTML pra desenvolvimento Web. É uma linguagem de estilos e formatações dos elementos da página de Internet desenvolvida, com propósito de formatar (estilizar) conteúdo HTML.

Explique UTF-8 e para que usamos.

É uma sigla definindo os caracteres para a língua Português Brasil, constando toda formatação correta dos mesmos, sendo assim não será exibido caracteres diferentes do idioma definido. Outros UTFs são para os caracteres de outros idiomas.
